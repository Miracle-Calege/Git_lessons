//andando entre commits/snapshots

git checkout numero do commit(e so usar o comando git log para saber o numero do commit)-> para ir para um determinado commit

ex:git checkout d44955b2884c50b08ee7faa0f75411c026609cc6
ex:git checkout d44955b(ou podemos usar somente os 7 primeiros digitos)

git checkout master-> vai para o ultimo commit

git revert-> reverte as mudanças feitas num determinado commit criando um novo sem as alterações.

git reset-> apaga todas as alterações feitas depois de um determinado commit.

NB:git reset pode ser usado com options como hard, soft e etc. Os comandos acima seguem a mesma estrutura do git checkout (precisam de ID depois do comando)
