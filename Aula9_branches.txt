branch->é uma espécie de workplace onde podemos fazer mudanças, testas novas ideias sem afectar o projecto inicial/principal.

git branch-> exibe todas as branch que temos.

//Para criar uma branch

git branch nome(a sua escolha)

ex:git branch teste

//git checkout-> também pode ser usado para nos levar de um branch para o outro,no lugar que colocávamos o ID do commit colocaremos o nome da branch

ex:git branch teste(saímos do branch main/master para o teste)

//Para deletar uma branch

git branch -d nome da branch(para que o comando funcione é importante que estejamos fora da branch que queremos apagar)

ex:git branch -d teste(Para um repositório que foi merged)

ex1:ex:git branch -D teste (Para um repositório que não foi merged)
